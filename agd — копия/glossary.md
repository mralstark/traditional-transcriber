# Глоссарий канонических написаний

Единый источник правды по терминам и именам. Используется для биасинга ASR
(`scripts/run_asr.py`, переменная `BIAS`) и для пост-коррекции в стадиях A/B.
Агент добавляет новые/спорные слова кандидатами, человек утверждает канон.

## Спикеры
- Константин Валерьевич **Малофеев** — бизнесмен, монархист, «Царьград».
- Александр Гельевич **Дугин** — философ-традиционалист.
- протоиерей Андрей Юрьевич **Ткачёв** — священник, проповедник. (пишем «Ткачёв», через «ё»)
